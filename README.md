# LinguaTrack — Веб-приложение для изучения иностранных языков

**LinguaTrack** — это современное геймифицированное веб-приложение для эффективного изучения иностранных языков и отслеживания прогресса. Проект разработан на основе клиент-серверной архитектуры с полной контейнеризацией всех компонентов.

---

## 🛠 Технологический стек

* **Frontend:** React, Vite, Tailwind CSS, Framer Motion
* **Backend:** Python 3, FastAPI (асинхронный фреймворк), Pydantic (валидация данных)
* **Database:** SQLite (встраиваемая база данных через ORM SQLAlchemy)
* **DevOps / Инфраструктура:** Docker, Docker Compose

---

## 🚀 Инструкция по развертыванию через Docker

Главное преимущество использования Docker — **абсолютная переносимость**. Приложение запустится идентично на любой ОС (Windows, macOS, Linux) без ручной установки Python, Node.js и зависимостей.

### Предварительные требования

1. **Git** (для клонирования репозитория) или архив с исходным кодом
2. **Docker** и **Docker Compose** (в современных версиях Docker встроен в `docker compose`)

### Пошаговый запуск

#### Шаг 1. Подготовка исходного кода

```bash
git clone https://github.com/HatsuneMikuLevan/LinguaTrack.git
cd LinguaTrack

#### Шаг 2. Сборка и запуск контейнеров
docker compose up --build

#### Шаг 3. Заполнение базы данных тестовыми данными (опционально)
docker compose exec backend python seed.py
